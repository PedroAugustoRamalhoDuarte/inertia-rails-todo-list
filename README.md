# Inertia Rails Todo List

- Basic repo for learning inertia with rails and make some proof of concept

## Techs

- Rails
- React
- Vite
- Inertia

## How to run

1. Clone the repo
2. Run `bundle install`
3. Run `yarn install`
4. Run `rails db:setup`
5. Run `./bin/dev`
